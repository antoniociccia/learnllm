# LLM Corso — Modulo 1: Dentro l'inferenza

Corso-gioco interattivo (IT/EN) sulla meccanica interna degli LLM: token, attention (Q/K/V),
KV cache, formula della memoria, budget VRAM, prefix caching, eviction. 7 capitoli + test a punti.

**Demo locale:** apri `index.html` in un browser. Zero dipendenze, zero build.

**Deploy:** è un file statico — GitHub Pages, Netlify, Vercel o un qualsiasi web server.
